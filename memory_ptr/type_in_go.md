   –ö–æ–≥–¥–∞ —É —Ç–µ–±—è –µ—Å—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–Ω–æ–≥–æ —Ç–∏–ø–∞ (interface{}) –∏ —Ç—ã –∑–Ω–∞–µ—à—å, –∫–∞–∫–æ–π —É –Ω–µ—ë —Ä–µ–∞–ª—å–Ω—ã–π —Ç–∏–ø,

2. Type switch
   –ï—Å–ª–∏ —Ç—ã –Ω–µ –∑–Ω–∞–µ—à—å –∑–∞—Ä–∞–Ω–µ–µ, –∫–∞–∫–æ–π —Ä–µ–∞–ª—å–Ω—ã–π —Ç–∏–ø —É –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–Ω–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å type switch

